# Code of Conduct

## Our Pledge

We are committed to creating a welcoming and inclusive environment for everyone.

## Expected Behavior

Please:
- be respectful
- be constructive
- support collaboration
- welcome newcomers

## Unacceptable Behavior

- harassment
- discrimination
- personal attacks
- trolling

## Enforcement

Project maintainers may remove comments, commits, issues, or contributions that violate this code of conduct.

---

Thank you for helping create a positive open-source community.

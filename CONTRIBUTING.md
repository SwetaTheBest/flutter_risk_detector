# Contributing to flutter_risk_detector

Thank you for contributing!

## 🚀 Getting Started

1. Fork the repository
2. Clone your fork
3. Create a new branch

```bash
git checkout -b feature/my-feature
```

4. Make changes
5. Run tests
6. Submit a pull request

---

## 📋 Code Style

Please:
- follow Dart formatting
- write clean commit messages
- add documentation when needed

Run:

```bash
flutter analyze
dart format .
```

before submitting.

---

## 🐛 Reporting Bugs

Please include:
- Flutter version
- Platform
- Reproduction steps
- Logs/screenshots if possible

---

## 💡 Feature Requests

Open an issue with:
- problem description
- expected behavior
- possible implementation idea

---

Thank you for helping improve flutter_risk_detector!
